# 0x01. C - Variables, if, else, while

## Description
In this project, you will learn about variables, conditionals, and loops in C programming. You will practice using `if`, `else`, `while`, and relational operators to create more complex programs.

## Learning Objectives
- How to use variables of different types.
- How to use `if` and `else` statements.
- How to use loops (`while`, `for`).
- The ASCII character set and how to manipulate it.

## Files and Scripts
| Filename             | Description |
|----------------------|-------------|
| `0-positive_or_negative.c`  | Prints whether a number is positive, negative, or zero. |
| `1-last_digit.c`      | Prints the last digit of a random number. |
| `2-print_alphabet.c`  | Prints the alphabet in lowercase. |
| `3-print_alphabets.c` | Prints the alphabet in lowercase and uppercase. |
| `4-print_alphabet.c`  | Prints the alphabet in lowercase without `q` and `e`. |
